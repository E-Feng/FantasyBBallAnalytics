## Getting Started (Client)

1. **Clone the repository:**
   ```sh
   git clone https://github.com/E-Feng/FantasyBBallAnalytics.git
   cd FantasyBBallAnalytics
   ```

2. **Install dependencies:**
   ```sh
   cd client
   npm install
   cd ..
   npm run dev
   ```
